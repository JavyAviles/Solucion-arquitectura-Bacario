# Solucion-arquitectura-Bacario
Arquitectura de sistema de banca virtual usando c4 model
Solución de arquitectura de banca virtual BP

## Contenido
1. Documento PDF con la solición a baca virtual BP
   -Diagrama C4
     -Diagrama de contexto del sistema
     -Diagrama de contenedores
     -Diagrama de componentes

## Arquitectura
Arquitectura basada en Clean Architecture, diseñada mediante Microservicios + Envent Driven + Cloud Native, siguiendo pricipios de:
-Alta cohesión 
-Bajo acoplamiento

## Tecnologías consideradas
Front-End:
-Angular y React Native (Web SPA)
-React Native y Flutter (Móvil)
Back-End:
-NodeJS
-Spring Boot
Infraestructura y monitoreo:
-AWS
-API Gateway
-SNS/SQS
-Kafka
-CloudWatch

## Seguridad
-Oauth 2.0
-TLS
-AWS Secret Manager
-Cumplimiento LODP y RGDP
